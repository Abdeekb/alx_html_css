# HTML & CSS Advanced Project

This project focuses on the application of advanced CSS concepts learned in the HTML and CSS course.

## Objective
- Build a web page with proper CSS styling based on the Figma design provided.
- Use CSS principles such as specificity, box model, and styling practices.
- No external libraries allowed (HTML, CSS, JavaScript only).

## Project Requirements
1. **HTML Structure:**
   - Build a valid HTML page using semantic tags.
   - The page should be structured and correctly display content as per the design in Figma.

2. **CSS Styling:**
   - Apply advanced CSS concepts to make the page visually appealing.
   - The CSS should include layout techniques, color schemes, typography, and any other necessary styling.
   - Ensure the page is responsive and works well on various screen sizes.

3. **Design:**
   - The design for this project is provided through Figma. You can access the Figma link by creating an account if necessary.
   - All fonts used in the design should be applied properly (e.g., Source Sans Pro, Spin-Cycle-OT).

4. **Important Notes:**
   - If the fonts used in the design are not installed on your system, you can download them from the links provided.
   - Some values in the design may be in float format, feel free to round them if needed.

5. **Images:**
   - All images for the project (such as logos) should be downloaded and included in the project folder.
   
6. **File Organization:**
   - Create a folder named `css_advanced` inside your project structure.
   - Place your `index.html` and any necessary CSS/JS files within this folder.

7. **Deliverables:**
   - You must submit a `README.md` file with detailed project information.
   - The project should include an `index.html` file, styled with CSS as per the provided design.
